    • What is JavaScript? And also Explain the history of JavaScript?

    JavaScript is a scripting or programming language that allows us to implement complex features on web pages. JavaScript is a lightweight, cross-platform, single-threaded, and interpreted compiled programming language. It is also known as the scripting language for webpages.

    History of JavaScript:

    JavaScript was invented by Brendan Eich in 1995. It was developed for Netscape 2, and became the ECMA-262 standard in 1997. Initially called Mocha, then LiveScript, it finally became known as JavaScript. It was designed for the client-side of websites to add dynamic and interactive elements to static HTML pages.
